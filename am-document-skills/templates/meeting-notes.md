# Template: Meeting Notes / Biên Bản Họp

## Structure

```
Header
├── Tiêu đề cuộc họp
├── Ngày: dd/MM/yyyy
├── Thời gian: HH:mm - HH:mm
├── Địa điểm / Link
└── Người ghi chép

Thành viên tham dự
├── [Tên] — [Chức vụ/Vai trò]  ✅ Có mặt
├── [Tên] — [Chức vụ/Vai trò]  ✅ Có mặt
└── [Tên] — [Chức vụ/Vai trò]  ❌ Vắng

Chương trình nghị sự (Agenda)
├── 1. [Chủ đề 1]
├── 2. [Chủ đề 2]
└── 3. [Chủ đề 3]

Nội dung thảo luận
├── Chủ đề 1
│   ├── Tóm tắt thảo luận
│   └── Kết luận
├── Chủ đề 2
│   ├── Tóm tắt thảo luận
│   └── Kết luận
└── Chủ đề 3

Quyết định
├── QĐ1: [Nội dung] — Approved by: [Tên]
└── QĐ2: [Nội dung] — Approved by: [Tên]

Action Items
┌──────┬────────────────┬──────────────┬────────────┬──────────┐
│ STT  │ Công việc      │ Người phụ    │ Deadline   │ Status   │
│      │                │ trách        │            │          │
├──────┼────────────────┼──────────────┼────────────┼──────────┤
│ 1    │ [Task]         │ [Tên]        │ dd/MM/yyyy │ Pending  │
│ 2    │ [Task]         │ [Tên]        │ dd/MM/yyyy │ Pending  │
└──────┴─���──────────────┴─────────��────┴────────────┴──��───────┘

Lịch họp tiếp theo
└── Ngày: dd/MM/yyyy — Nội dung dự kiến
```

## Specs

| Property | Value |
|----------|-------|
| Format | DOCX (dễ share & edit) |
| Page | A4, 1-3 trang |
| Font | Arial 11pt |
| Action items | PHẢI có table với người phụ trách + deadline |
| Tone | Ngắn gọn, bullet points, không narrative |

## Khi Nào Dùng

- Biên bản họp team
- Meeting minutes
- Kick-off notes
- Standup / Sprint review notes
